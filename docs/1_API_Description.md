# Описание API — AeroGuys

## Общие сведения

**Базовый URL:** `http://localhost:8000`  
**Версия:** `1.0.0`  
**Формат данных:** JSON (запросы и ответы)  
**Интерактивная документация:** `http://localhost:8000/api/docs` (Swagger UI), `http://localhost:8000/api/redoc` (ReDoc)

API построен на базе **FastAPI** и предоставляет аналитические данные об авиационном трафике, собранных из OpenSky Network. Все конечные точки доступны через `GET` или `POST` запросы.

---

## Аутентификация

Для работы API аутентификация на уровне запросов не требуется. Учётные данные OpenSky Network (`OPENSKY_CLIENT_ID`, `OPENSKY_CLIENT_SECRET`) настраиваются на стороне сервера через переменные окружения.

---

## Модули API

| Префикс            | Модуль              | Назначение                              |
|--------------------|---------------------|-----------------------------------------|
| `/api/init`        | Инициализация       | Загрузка данных из OpenSky и из CSV     |
| `/api/realtime`    | Реальное время      | Мониторинг воздушного трафика           |
| `/api/airports`    | Аэропорты           | Аналитика аэропортов                    |
| `/api/aircraft`    | Воздушные суда      | Аналитика ВС и позиций                  |
| `/api/routes`      | Маршруты            | Аналитика маршрутов и авиакомпаний      |
| `/api/export`      | Экспорт             | Скачивание данных в CSV/ZIP             |

## Эндпоинты

### Инициализация (`/api/init`)

#### `GET /api/init/status`
Проверить наличие данных в базе данных.

**Ответ `200`:**
```json
{
  "has_data": true,
  "flights": 15234,
  "states": 48921
}
```

---

#### `POST /api/init/realtime`
Выполнить один цикл сбора данных из OpenSky Network: текущие позиции (state vectors) и рейсы за последние 2 часа.

**Ответ `200`:**
```json
{
  "states_loaded": 8452,
  "flights_loaded": 1203,
  "message": "OK"
}
```

**Ответ `503`:** Учётные данные OpenSky API не настроены.

---

#### `POST /api/init/upload-csv`
Загрузить рейсы из пользовательского CSV-файла.

**Тело запроса:** `multipart/form-data` — поле `file` (CSV-файл).

**Ответ `200`:** Аналогично `/api/init/realtime`.

---

### Реальное время (`/api/realtime`)

#### `GET /api/realtime/airport-busyness`
Рейтинг загруженности аэропортов по суммарному числу рейсов.

| Параметр    | Тип     | Обязательный | По умолчанию | Описание                        |
|-------------|---------|:---:|------|---------------------------------|
| `hours_back`| integer | —   | 24   | Глубина выборки в часах (1–168) |
| `limit`     | integer | —   | 20   | Максимум записей (1–100)        |

**Ответ `200`:** Массив объектов:
```json
[
  {
    "airport": "EDDF",
    "departures": 312,
    "arrivals": 298,
    "total_flights": 610,
    "first_seen": 1714521600,
    "last_seen": 1714608000
  }
]
```

---

#### `GET /api/realtime/city-busyness`
Рейтинг загруженности городов (аэропорты одного города суммируются).

| Параметр    | Тип     | Обязательный | По умолчанию |
|-------------|---------|:---:|------|
| `hours_back`| integer | —   | 24   |
| `limit`     | integer | —   | 20   |

---

#### `GET /api/realtime/fastest`
Топ самых быстрых воздушных судов из последнего снапшота (только в воздухе).

| Параметр | Тип     | По умолчанию |
|----------|---------|------|
| `limit`  | integer | 20   |

**Ответ `200`:** Массив `AircraftPosition`.

---

#### `GET /api/realtime/highest`
Топ воздушных судов по барометрической высоте из последнего снапшота.

| Параметр | Тип     | По умолчанию |
|----------|---------|------|
| `limit`  | integer | 20   |

---

### Аэропорты (`/api/airports`)

#### `GET /api/airports/stats`
Рейтинг аэропортов по загруженности за всё время или за N дней.

| Параметр | Тип     | По умолчанию | Описание               |
|----------|---------|------|------------------------|
| `days`   | integer | 0    | 0 — весь доступный снапшот |

---

#### `GET /api/airports/throughput`
Средняя пропускная способность (рейсов/час) для набора аэропортов.

| Параметр   | Тип           | Обязательный | По умолчанию |
|------------|---------------|:---:|------|
| `airports` | array[string] | ✓   | —    |
| `days`     | integer       | —   | 7    |

Пример запроса: `GET /api/airports/throughput?airports=EDDF&airports=LFPG&days=7`

---

#### `GET /api/airports/{icao}/daily-trend`
Ежедневная динамика вылетов, прилётов и суммы рейсов за период.

| Параметр | Расположение | Обязательный |
|----------|-------|:---:|
| `icao`   | path  | ✓   |
| `days`   | query | —   |

---

#### `GET /api/airports/{icao}/destinations`
Куда и сколько раз вылетали самолёты из данного аэропорта.

| Параметр | По умолчанию |
|----------|------|
| `icao`   | —    |
| `days`   | 7    |

---

#### `GET /api/airports/{icao}/info`
Координаты, страна и город аэропорта из справочника.

| Параметр | Расположение | Обязательный |
|----------|-------|:---:|
| `icao`   | path  | ✓   |

---

#### `GET /api/airports/{icao}/peak-hours`
Почасовое распределение вылетов и прилётов.

| Параметр | По умолчанию |
|----------|------|
| `icao`   | —    |
| `days`   | 7    |

---

### Воздушные суда (`/api/aircraft`)

#### `GET /api/aircraft/positions`
Текущие позиции всех бортов из последнего снапшота с опциональной фильтрацией.

| Параметр       | Тип     | Описание                       |
|----------------|---------|--------------------------------|
| `min_lat`      | float   | Нижняя граница широты (-90..90)|
| `max_lat`      | float   | Верхняя граница широты         |
| `min_lon`      | float   | Левая граница долготы          |
| `max_lon`      | float   | Правая граница долготы         |
| `min_altitude` | float   | Минимальная высота, м          |
| `on_ground`    | boolean | `true` — на земле, `false` — в воздухе |
| `limit`        | integer | До 5000, по умолчанию 1000     |

**Ответ `200`:** Массив `AircraftPosition`:
```json
[
  {
    "icao24": "3c6444",
    "callsign": "DLH123",
    "latitude": 50.033,
    "longitude": 8.570,
    "altitude_m": 9850.0,
    "speed_kmh": 892.5,
    "vertical_rate_ms": 0.0,
    "on_ground": false,
    "last_contact": 1714608234,
    "origin_country": "Germany"
  }
]
```

---

#### `GET /api/aircraft/by-callsign/{callsign}`
Найти конкретный борт по позывному в последнем снапшоте.

**Ответ `404`:** Борт не найден.

---

#### `GET /api/aircraft/types`
Распределение воздушных судов по категориям ADS-B за последние N дней.

| Параметр  | По умолчанию |
|-----------|------|
| `days`    | 7    |
| `min_lat`, `max_lat`, `min_lon`, `max_lon` | Опционально |

---

#### `GET /api/aircraft/altitude-profile`
Средняя и медианная крейсерская высота и скорость по каждой категории ВС.

---

#### `GET /api/aircraft/altitude-distribution`
Гистограмма высот: 10 бакетов от 0 до 15 000 м.

---

#### `GET /api/aircraft/speed-distribution`
Гистограмма скоростей воздушных судов из последнего снапшота.

---

#### `GET /api/aircraft/country-distribution`
Топ стран по числу уникальных ВС в последнем снапшоте.

| Параметр | По умолчанию |
|----------|------|
| `limit`  | 15   |

---

#### `GET /api/aircraft/snapshot-stats`
Агрегированная статистика последнего снапшота: всего бортов, в воздухе, на земле и т.д.

---

#### `GET /api/aircraft/flight-phases`
Фазы полёта из последнего снапшота: набор высоты, снижение, горизонтальный полёт.

---

#### `GET /api/aircraft/unidentified`
Военные и неопознанные борты (без позывного или с нестандартным позывным).

| Параметр | По умолчанию |
|----------|------|
| `limit`  | 50   |

---

#### `GET /api/aircraft/business`
Бизнес-авиация: борты категорий Light/Small/High-performance с нестандартными позывными.

| Параметр | По умолчанию |
|----------|------|
| `limit`  | 100  |

---

#### `GET /api/aircraft/extreme-vertical-rates`
Борты с резкими наборами или снижениями высоты.

---

#### `GET /api/aircraft/{icao24}/usage`
Ежедневный налёт конкретного воздушного судна.

---

#### `GET /api/aircraft/{icao24}/routes`
Типичные маршруты конкретного воздушного судна.

### Маршруты (`/api/routes`)

#### `GET /api/routes/popular`
Самые частотные пары аэропортов за выбранный период.

| Параметр | По умолчанию |
|----------|------|
| `days`   | 7    |
| `limit`  | 20   |

**Ответ `200`:**
```json
[
  {
    "departure_airport": "EDDF",
    "arrival_airport": "LFPG",
    "flight_count": 42
  }
]
```

---

#### `GET /api/routes/efficiency`
Сравнение фактической длительности рейса с теоретической ортодромией.

| Параметр | По умолчанию |
|----------|------|
| `days`   | 7    |
| `limit`  | 20   |

---

#### `GET /api/routes/duration-distribution`
Гистограмма длительностей рейсов: 12 бакетов по 60 мин (0–720 мин).

| Параметр | По умолчанию |
|----------|------|
| `days`   | 7    |

---

#### `GET /api/routes/airlines`
Топ авиакомпаний по числу рейсов (по коду ICAO из позывного).

| Параметр | По умолчанию |
|----------|------|
| `days`   | 7    |
| `limit`  | 15   |

---

### Экспорт (`/api/export`)

#### `GET /api/export/flights`
Скачать CSV-файл с рейсами за указанный период.

| Параметр | Тип      | Обязательный | Описание                              |
|----------|----------|:---:|---------------------------------------|
| `start`  | datetime | ✓   | Начало периода (ISO: `2026-03-01T00:00:00`) |
| `end`    | datetime | ✓   | Конец периода                         |

**Ответ `200`:** CSV-файл (`Content-Disposition: attachment; filename=flights_...csv`).  
Столбцы: `ICAO24`, `Callsign`, `Departure Time`, `Arrival Time`, `From`, `To`, `Duration (min)`.

---

#### `GET /api/export/raw`
Скачать все сырые данные о рейсах без фильтра по дате (все столбцы).

---

#### `GET /api/export/analytics`
Скачать ZIP-архив с аналитическими CSV (статистика аэропортов, маршруты, ВС и т.д.).

---

## Коды ошибок

| Код  | Описание                                      |
|------|-----------------------------------------------|
| 200  | Успешный ответ                                |
| 404  | Ресурс не найден (например, борт по позывному)|
| 422  | Ошибка валидации параметров запроса           |
| 503  | Сервис недоступен (нет учётных данных OpenSky)|
