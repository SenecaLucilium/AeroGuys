## Аэропорты


### `GET /api/airports/stats`

**Рейтинг аэропортов по загруженности**

Статистика всех аэропортов. days=0 — весь доступный снапшот (all-time).


**Параметры запроса:**

| Имя | В | Тип | Обязательный | Описание |
|-----|---|-----|:---:|---------|
| `days` | query | integer (default: 0) | — |  |

**Коды ответов:**

- **200** — Successful Response
- **422** — Validation Error


### `GET /api/airports/throughput`

**Пропускная способность аэропортов**

Среднее число рейсов в час для указанных аэропортов. Передайте несколько значений параметра `airports` для сравнения.


**Параметры запроса:**

| Имя | В | Тип | Обязательный | Описание |
|-----|---|-----|:---:|---------|
| `airports` | query | array | ✓ | ICAO-коды аэропортов, например: airports=EDDF&airports=LFPG |
| `days` | query | integer (default: 7) | — |  |

**Коды ответов:**

- **200** — Successful Response
- **422** — Validation Error

## Инициализация


### `POST /api/init/realtime`

**Загрузить real-time данные из OpenSky API**

Выполняет один цикл сбора данных из OpenSky Network:
state vectors (текущие позиции) + рейсы за последние 2 часа.


**Коды ответов:**

- **200** — Successful Response


### `GET /api/init/status`

**Проверить наличие данных в БД**

Возвращает, есть ли в базе загруженные данные.


**Коды ответов:**

- **200** — Successful Response


### `POST /api/init/upload-csv`

**Загрузить рейсы из CSV-файла**

Принимает CSV с заголовком (регистр не важен):
  ICAO24, Callsign, Departure Time, Arrival Time, From, To, Duration (min)

Дубликаты пропускаются автоматически (ON CONFLICT DO NOTHING).


**Коды ответов:**

- **200** — Successful Response
- **422** — Validation Error

## Экспорт


### `GET /api/export/analytics`

**Выгрузить всю аналитику в ZIP-архиве (несколько CSV)**

Собирает аналитические выборки из БД и упаковывает в ZIP-архив:
  • airport_statistics.csv  — трафик по аэропортам
  • top_routes.csv          — топ-маршруты
  • aircraft_activity.csv   — активность воздушных судов
  • country_distribution.csv — распределение по странам (state vectors)


**Коды ответов:**

- **200** — Successful Response


### `GET /api/export/flights`

**Экспорт рейсов в CSV**

Скачивает CSV-файл со всеми рейсами за указанный период.


**Параметры запроса:**

| Имя | В | Тип | Обязательный | Описание |
|-----|---|-----|:---:|---------|
| `start` | query | string | ✓ | Начало периода, ISO-формат: 2026-03-01T00:00:00 |
| `end` | query | string | ✓ | Конец периода, ISO-формат: 2026-03-21T23:59:59 |

**Коды ответов:**

- **200** — Successful Response
- **422** — Validation Error


### `GET /api/export/raw`

**Выгрузить все сырые данные о рейсах в CSV**

Скачивает все рейсы из БД со всеми столбцами (без фильтра по дате).


**Коды ответов:**

- **200** — Successful Response
